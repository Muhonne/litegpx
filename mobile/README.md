## Stack

- Kotlin
- Jetpack Compose
- Gradle wrapper
- MapLibre Android SDK
- Google Play Services location

Code starting points:

- `app/src/main/java/com/example/traillite/MainActivity.kt` for app state, Compose UI, route selection, import actions, and settings.
- `app/src/main/java/com/example/traillite/TrailMapController.kt` for MapLibre setup, route overlays, location dot, route projection, and navigation camera.
- `app/src/main/java/com/example/traillite/TrailStorage.kt` for local file storage, map package selection, style generation, and provider overlay injection.
- `app/src/main/java/com/example/traillite/GpxParser.kt` for GPX track-point parsing.
- `app/src/main/java/com/example/traillite/BundledRoute.kt` for bundled route catalog loading.
- `app/src/main/java/com/example/traillite/BatteryLocationClient.kt` for location update cadence and permissions.
- `app/src/main/java/com/example/traillite/MapLayerSettings.kt` for persisted map/settings state.

## Build

From this directory:

```sh
JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home" ./gradlew assembleDebug
```

The debug APK is written to:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Install And Launch

With an emulator or device attached:

```sh
/Users/mattias/Library/Android/sdk/platform-tools/adb devices
```

From the workspace root, the install/launch command used during local testing is:

```sh
/Users/mattias/Library/Android/sdk/platform-tools/adb install -r mobile/app/build/outputs/apk/debug/app-debug.apk
/Users/mattias/Library/Android/sdk/platform-tools/adb shell am force-stop com.example.traillite
/Users/mattias/Library/Android/sdk/platform-tools/adb shell monkey -p com.example.traillite -c android.intent.category.LAUNCHER 1
```

## Bundled Assets

The Android build merges app assets from:

- `app/src/main/assets/`
- `../shared/`

The merge is configured in `app/build.gradle.kts` under `sourceSets`.

Useful APK inspection:

```sh
zipinfo -1 app/build/outputs/apk/debug/app-debug.apk | rg 'assets/(maps|routes|styles)'
```

Expected optional map assets when generated locally:

- `assets/maps/finland.pmtiles`
- `assets/maps/finland.providers.pmtiles`
- `assets/maps/manifest.json`
- `assets/styles/style_template.json`

`TrailStorage.ensureBundledMapPackage` copies bundled map assets into app-specific storage before MapLibre loads them. When `assets/maps/manifest.json` changes, the app refreshes both copied PMTiles files so a newly installed build uses the route corridor and provider detail generated by the web save flow.

